🖥️ Prodooh Player Setup

Configura este browser como un dispositivo player (simula una Raspberry Pi)

¿Cómo funciona?
1. Ingresa el Venue ID y Device Token (8 caracteres, se muestra al registrar la pantalla)
2. Al guardar, se autentica contra el backend con el device_token
3. Si la autenticación es exitosa, abre el player en pantalla completa
